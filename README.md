Overview

The main deliverable of this repository is a PDF file with test cases covering three key end-to-end user flows in Qodana Cloud:

Flow 1: Creating an Organization and Inviting a Member
Flow 2: Setting Up a New Project and Linking it to a Repository
Flow 3: Running a Code Quality Analysis and Reviewing Results

Contents
1. QodanaCloud_TestCases.pdf

- Descriptions of the selected user flows
- Test cases in table format
- Notes and suggestions based on real product usage

2. qodana_code_quality.yml

- Configuration file for Qodana code quality checks.

3. qodana.yml

- Qodana project configuration file, used for setting up and running code analysis.

How to Use

1. Review the PDF for a comprehensive understanding of Qodana Cloud’s main user flows and the associated test cases.
2. Use the YAML files as reference or templates for configuring Qodana in your own projects.
3. Each test case is designed to be actionable and easy to follow for manual testing or future automation.
